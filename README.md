# Multi-Factor ROI Optimization: Colorado Real Estate Dynamics
**Author:** Barsha Kakshapati  
**Institution:** Regis University | M.S. Data Science  

## Project Overview
This practicum develops a machine learning framework to predict 3-year Return on Investment (ROI) for residential real estate in Colorado. By integrating Zillow pricing data with socio-economic indicators from the U.S. Census and school density metrics from the NCES, the model identifies "safety floors" that stabilize property values.

## Key Features
* **Random Forest Regressor:** Optimized ensemble model with an $R^2$ of 0.85.
* **Infrastructure Analysis:** Quantifies the impact of school density on capital appreciation.
* **Risk Mitigation:** Uses Mean Absolute Error (MAE) of 7.24% as a signal-to-noise threshold for investment decisions.

## Repository Structure
* `/data`: Preprocessed datasets (Zillow, Census, NCES).
* `/notebooks`: Jupyter notebooks for EDA and Model Training.
* `/src`: Python scripts for the data pipeline.
* `Practicum_Report.pdf`: Final academic documentation.
